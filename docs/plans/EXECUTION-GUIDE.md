## 故障排查

如果遇到问题：

### 常见问题1: 找不到计划文件
**解决方案**: 确保在新会话中工作目录是项目根目录

### 常见问题2: 测试失败
**解决方案**: 检查依赖是否正确安装，运行 `pip install -r requirements.txt`

### 常见问题3: 图表不显示
**解决方案**: 检查浏览器控制台，可能是CDN资源加载失败，检查网络连接

### 常见问题4: 数据未注入
**解决方案**: 确保 `window.dashboardData` 在HTML中正确设置
